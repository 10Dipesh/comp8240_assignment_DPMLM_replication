## Project Overview

This repository contains the full replication of the paper:

> **“DP-MLM: Differentially Private Text Rewriting Using Masked Language Models”**  
> _Stephen Meisenbacher, Maulik Chevli, Juraj Vladika, and Florian Matthes_  
> Findings of ACL 2024. DOI: [10.18653/v1/2024.findings-acl.554](https://doi.org/10.18653/v1/2024.findings-acl.554)

The goal of this project is to reproduce the results of the original DP-MLM paper on **GLUE benchmark tasks** (CoLA, MRPC, RTE, SST-2) under three differential-privacy budgets  
(ε = 10, 50, 250), and extend the framework to **new cross-domain datasets** — including **IMDb Reviews**, **Mini-Yelp**, and **Name-Redacted Bios**.
